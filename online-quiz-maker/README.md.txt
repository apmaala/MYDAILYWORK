# Job Board Platform – Full Stack Application

A full-stack Job Board platform where Employers can post jobs, Candidates can apply with resumes, and Admin can manage the system.

---

## 🚀 Tech Stack

Frontend:
- React.js (Vite)
- Tailwind CSS
- Context API
- Recharts

Backend:
- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT Authentication
- Multer (Resume Upload)
- Helmet, CORS, Rate Limiting

---

## 👥 User Roles

Admin:
- Manage users
- Manage jobs
- View platform statistics

Employer:
- Post jobs
- View applications
- See analytics dashboard

Candidate:
- Register & Login
- Search jobs
- Apply with resume
- Track application status

## ⚙️ Installation

1. Clone the repository
2. Install backend dependencies:
   npm install

3. Create .env file:

PORT=5000  
MONGO_URI=your_mongodb_connection_string  
JWT_SECRET=your_secret_key  

4. Run backend:
   npm run dev

5. Install frontend dependencies:
   npm install

6. Run frontend:
   npm run dev
